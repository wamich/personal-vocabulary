# v0.13

## 0.13.2

- **优化**：交互细节调整

- **修复**：YT 字幕高亮失效的问题

## 0.13.1

- **更新**：

  1. **预留短语固定搭配**（目前没有数据，暂不不展示，服务端能分析短语数据时自动展示）
  2. **完善朗读**功能，在朗读**句子**模式下，按 W、X 键，朗读前一句、后一句
  3. **更新项目底层依赖**，同时 UI 有细微的变化
  4. **取消安卓 Kiwi 推荐**。原因：Kiwi 长时间不更新，更不上新需求，同时 Edge 更简便、能自动更新插件。

- **修复**：

  1. 增加生词**不能高亮**的边界情况
  2. W、X 键，在前后导航时，无法**自动滚动到视野**中
  3. **设置页** -> **高亮规则** -> **新增规则**中，表单校验错误

## 0.13.0

1. 查词界面中，增加 **AI 词典**（将持续完善...）；
2. 改进 **兼容模式**，大部分情况，不再需要额外设置；
3. **PDF** 高亮**仅支持: <u>下划线</u> 高亮**，**不支持:** <del>背景+字体色</del>；
4. 点击插件图标**弹出页**中，【文件】增加多网站支持，如： [沉浸式翻译](https://app.immersivetranslate.com/)、[Koodo](https://web.koodoreader.com/)、[Flow](https://app.flowoss.com/)；
5. 修复 **与简阅插件适配 bug** [@Qin-Dong](https://gitee.com/wamich/personal-vocabulary/issues/IBFVEC)；
6. 完善 **R 键朗读** 功能。 感谢 B 站 <u>窘鸭</u>、<u>鬥公举 fuam</u> 建议

   - 当查词或翻译时，朗读单词或整句；
   - 当有选择文本时，朗读选中内容；
   - 除了上述 2 种情况，朗读鼠标所指内容。
     - 只按 **1 次**，朗读**单词**
     - 连按 **2 次**，朗读**整句**
